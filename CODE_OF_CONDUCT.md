# Code of Conduct

## Our pledge

We want lanner to be a welcoming project for everyone, regardless of background
or experience. We pledge to treat all participants with respect.

## Expected behaviour

- Be respectful, considerate, and constructive.
- Welcome newcomers and assume good intent.
- Accept constructive feedback gracefully.

## Unacceptable behaviour

- Harassment, insults, or discriminatory language or imagery.
- Personal or political attacks.
- Publishing others' private information without consent.

## Enforcement

Report unacceptable behaviour to the maintainer through the contact on the
GitHub profile, or via private vulnerability reporting for sensitive matters.
Maintainers may remove comments, commits, issues, or contributions that violate
this code, and may ban repeat or severe offenders.

## Scope

This code applies in all project spaces (the repository, issues, pull requests,
and discussions) and when representing the project in public spaces.
